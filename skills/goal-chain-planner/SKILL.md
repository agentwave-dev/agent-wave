# goal-chain-planner

Use this skill when a roadmap item needs to become a Goal Chain Wave.

Steps:

1. Copy `templates/goal-chain.yaml`.
2. Rewrite the objective as a bounded outcome.
3. Split the objective into small ordered goals.
4. Assign one lane and one primary skill per goal.
5. Add validation commands for every goal.
6. Require receipts before advancement.
7. Set the merge gate to `human-review-required`.
8. Avoid goals that require private data, secrets, or raw runtime logs.
