# learning-ledger

Use this skill when a completed run produced a reusable operational lesson.

Steps:

1. Copy `templates/learning-ledger-entry.md`.
2. Link the entry to a lane, goal, trace, and evidence receipt.
3. Write one concrete observation.
4. Write one decision derived from that observation.
5. Write one future instruction for agents.
6. Set promotion status to `local-note`, `candidate-skill`, or `promoted`.
7. Do not include secrets, raw logs, private payloads, or unreviewed speculation.
